# Evolution Logs

This directory tracks the evolution of AI agent capabilities, features, and behaviors over time.

## Purpose

Evolution logs document:
- New capabilities added to agents
- Improvements in reasoning or problem-solving
- Changes in behavior patterns
- Learning milestones
- Adaptations and optimizations

## Log Format

Each log entry should follow this format:

```markdown
# Evolution Log Entry

**Date**: YYYY-MM-DD HH:MM:SS UTC
**Version**: X.Y.Z
**Type**: [Capability|Behavior|Optimization|Learning]

## Summary
Brief description of the evolution

## Details
Detailed explanation of what changed and why

## Impact
Expected impact on agent performance or capabilities

## Related Issues/PRs
- Issue #XXX
- PR #YYY
```

## Example

See `example-evolution-log.md` for a sample entry.

## Naming Convention

Files should be named: `YYYY-MM-DD-brief-description.md`

Example: `2026-01-10-improved-context-understanding.md`
